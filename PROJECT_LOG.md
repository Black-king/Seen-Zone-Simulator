# Seen Zone Simulator 项目记录

> 记录当前已完成的升级、阶段成果，以及下一步计划。

## 项目定位

**Seen Zone Simulator / 已读不回模拟器** 是一个把“已读不回焦虑”做成互动网页玩具的开源项目。
目标不是解决社交焦虑，而是把这种体验具象化、荒诞化、可分享化。

## 已完成的升级

### 1. 从点子升级为可运行产品
- 已从纯概念，升级为可启动、可交互、可部署的前端 MVP。
- 已完成基础架构搭建，项目可以本地运行并构建通过。

### 2. 从单一聊天效果升级为完整模拟器
- 支持输入对方名字。
- 支持选择关系模式：
  - 暧昧对象
  - 前任
  - 老板
  - 甲方
  - 朋友
- 支持发送消息后模拟：
  - 正在输入...
  - 已读
  - 无回复
  - 随机撤回消息
- 增加了焦虑指数、已读时长、脑补次数等拟真指标。

### 3. 从普通页面升级为有氛围的作品
- 视觉上采用深夜暗色、霓虹强调、玻璃拟态、轻故障感。
- 加入浮动光球、输入动画、气泡动效、弹窗报告等反馈。
- 移动端也做了适配，适合传播和截图。

### 4. 从单页演示升级为开源项目形态
- 已补齐 GitHub Pages 自动部署工作流。
- 已补齐 README、LICENSE、CONTRIBUTING 等开源文件。
- 已生成 `package-lock.json`，方便稳定安装与 CI 构建。

### 6. 从展示型页面升级为微信聊天工具风格
- 将主界面重做为更像微信的聊天工具：顶部栏、聊天区、消息气泡、底部输入栏更贴近真实沟通软件。
- 增加更多随机事件，强化“已读不回”的戏剧性和传播性。
- 增加 Web Audio 合成音效和音效开关，不依赖外部音频素材。

### 7. 从聊天内提示升级为侧边安抚区
- 将原本嵌在聊天流里的绿色安抚提示移出对话区，改放到左侧控制面板。
- 让聊天窗口保持更干净的微信式视觉，减少“演示组件感”。
- 侧边栏现在承担状态展示与辅助提示，主聊天区只保留真正的对话内容。

## 下一步计划

### 优先级 P0：传播能力
1. **分享海报 / 截图卡片**
   - 一键生成“已读不回报告”卡片
   - 便于发社交平台

2. **更强的结果页**
   - 结尾总结更荒诞、更像社交诊断单
   - 支持复制文案

3. **更多随机事件**
   - 对方上线又下线
   - 对方撤回半句话
   - 对方输入后消失
   - “稍后回复”的幽灵提示

### 优先级 P1：趣味性增强
4. **音效系统**
   - 正在输入音
   - 已读提示音
   - 撤回提示音

5. **更多模式皮肤**
   - 深夜焦虑风
   - 糖果整蛊风
   - 极简黑白风
   - 社死红色警报风

6. **更丰富的关系模式**
   - 室友
   - 同事
   - 亲戚
   - 领导
   - 客户群聊

### 优先级 P2：产品化
7. **PWA 支持**
   - 可安装到桌面/手机
   - 更像一个真正的“工具/玩具”

8. **多语言支持**
   - 中文
   - English

9. **数据埋点或统计**
   - 常见发送语句
   - 最常用关系模式
   - 最容易破防的节点
