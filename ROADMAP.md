# Roadmap

> Seen Zone Simulator 的路线图。重点方向：更好玩、更好看、更容易传播。

---

## 当前阶段：MVP 已完成

项目已经具备完整的基础体验：

- 可输入名字
- 可选择关系模式
- 可发送消息
- 可模拟正在输入
- 可模拟已读不回
- 可随机触发撤回消息
- 可生成焦虑报告
- 可部署到 GitHub Pages

---

## Phase 1：传播增强

目标：让用户更愿意截图、转发、整蛊朋友。

### 1.1 分享海报

- [x] 将“已读不回报告”导出为图片。
- [x] 增加海报模板：
  - 深夜焦虑风
  - 社死诊断单
  - 聊天截图风
- [x] 支持复制图片或下载 PNG。

### 1.2 分享文案优化

- [ ] 增加更多结论文案。
- [ ] 根据关系模式生成不同报告语气。
- [ ] 根据焦虑指数生成不同等级称号：
  - 轻微在意
  - 反复打开
  - 标点分析师
  - 已读重灾区
  - 尊严离线

### 1.3 一键重开

- [ ] 增加“再被已读一次”按钮。
- [ ] 增加随机默认消息。
- [ ] 增加“今天不适合聊天”彩蛋。

---

## Phase 2：趣味增强

目标：让模拟器更像一个荒诞小游戏。

### 2.1 更多随机事件

- [ ] 对方上线又离线。
- [ ] 对方正在输入 0.5 秒后消失。
- [ ] 对方正在输入 10 秒，最后什么都没有。
- [ ] 对方撤回半句话。
- [ ] 对方点赞了朋友圈，但没回消息。
- [ ] 对方显示“语音转文字中”，然后消失。

### 2.2 更多关系模式

- [ ] 同事
- [ ] 室友
- [ ] 亲戚
- [ ] 导师
- [ ] 客户群聊
- [ ] HR
- [ ] 网友
- [ ] 游戏队友

### 2.3 连发惩罚

- [ ] 连发 3 条：提示“你正在失去最后的体面”。
- [ ] 连发 5 条：触发疗愈提示。
- [ ] 连发 8 条：聊天框轻微抖动。
- [ ] 连发 10 条：系统建议“先退出聊天窗口”。

---

## Phase 3：视听体验

目标：把“已读不回”的情绪做得更有沉浸感。

### 3.1 音效

- [ ] 消息发送音。
- [ ] 正在输入提示音。
- [ ] 已读提示音。
- [ ] 撤回提示音。
- [ ] 焦虑指数接近 100% 时的低频氛围音。
- [ ] 音效开关与静音记忆。

### 3.2 主题皮肤

- [ ] 深夜焦虑风。
- [ ] 糖果社死风。
- [ ] 极简黑白风。
- [ ] 故障霓虹风。
- [ ] 仿聊天软件皮肤。

### 3.3 动效增强

- [ ] 已读出现时轻微闪烁。
- [ ] 撤回消息模糊坍缩。
- [ ] 焦虑指数过高时背景轻微变暗。
- [ ] 输入状态气泡更拟真。

## 推荐下一步

如果目标是快速传播，优先做：

1. 分享海报导出
2. 更多破防文案
3. 一键重开
4. GitHub Pages 正式发布

如果目标是打磨作品集，优先做：

1. 动效增强
2. 主题皮肤
3. PWA
4. 多语言
