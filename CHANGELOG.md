## [0.1.0] - 2026-07-19

### Added

- 初始化项目：`Seen Zone Simulator / 已读不回模拟器`。
- 使用 Vite + React + TypeScript 搭建前端项目。
- 实现基础聊天模拟流程：
  - 用户输入消息。
  - 点击发送。
  - 对方显示“正在输入...”。
  - 约 3 秒后输入状态消失。
  - 显示“已读”。
  - 进入无回复沉默状态。
- 增加关系模式：
  - 暧昧对象
  - 前任
  - 老板
  - 甲方
  - 朋友
- 不同关系模式拥有独立：
  - 默认名字
  - 焦虑基准值
  - 文案池
  - 撤回文案
  - 假回复片段
  - 主题强调色
- 增加随机事件：
  - 对方再次正在输入
  - 对方撤回一条消息
  - 对方输入后再次沉默
- 增加焦虑指数系统：
  - 基于关系模式、发送次数、已读时长动态增长。
  - 深夜模式下焦虑增长加成。
- 增加统计指标：
  - 已读时长
  - 连发消息数
  - 脑补次数
  - 尊严剩余
- 增加“已读不回报告”弹窗。
- 支持复制报告文案，方便整蛊和分享。
- 增加移动端响应式布局。
- 增加深夜暗色、霓虹、玻璃拟态、轻故障感视觉风格。
- 增加 GitHub Pages 自动部署工作流。
- 增加基础开源文档：
  - `README.md`
  - `LICENSE`
  - `CONTRIBUTING.md`
  - `PROJECT_LOG.md`
  - `CHANGELOG.md`
  - `ROADMAP.md`

### Changed

- 将项目从“vibecoding 点子”升级为可运行、可构建、可部署的 MVP。
- 将依赖版本从 `latest` 调整为稳定版本范围。
- 将构建工具移动到 `devDependencies`，保持依赖结构更清晰。
- 修正 TypeScript 配置以兼容当前 TypeScript/Vite 构建流程。

### Fixed

- 修复重置聊天后旧异步流程可能继续写入消息的问题。
- 修复第 5 次发送时“疗愈/破防”分支触发不准确的问题。
- 修复构建缓存文件进入 Git 跟踪的问题，已在 `.gitignore` 中忽略 `*.tsbuildinfo`。

## [0.1.1] - 2026-07-19

### Added

- 增加“下载分享海报”能力。
- 报告弹窗增加右侧海报预览。
- 分享入口更适合截图、转发和社交传播。

### Changed

- 报告弹窗由单栏升级为双栏布局。
- 传播文案与结果展示更加适合二次分享。

## [0.1.2] - 2026-07-19

### Added

- 增加更多随机事件：
  - 对方短暂正在输入后消失。
  - 对方长时间正在输入但没有回复。
  - 对方上线又下线。
  - 对方点赞朋友圈但不回消息。
  - 对方显示语音转文字后消失。
  - 对方撤回半句话。
  - 消息进入“稍后再说”队列。
- 增加 Web Audio 合成音效系统：
  - 发送音。
  - 正在输入提示音。
  - 已读提示音。
  - 撤回提示音。
  - 上线/动态提示音。
  - 焦虑警报音。
  - 疗愈提示音。
- 增加音效开关，并使用 `localStorage` 记忆用户选择。

### Changed

- 将界面视觉重做为微信聊天工具风格。
- 聊天窗口成为主视觉，减少原先 landing page 的展示感。
- 重做消息气泡、顶部聊天栏、底部输入栏、系统提示、控制面板样式。
- 报告弹窗适配新的浅色微信式视觉体系。

### Fixed

- 改善原先排版重心不清晰、缺少沟通工具感的问题。
